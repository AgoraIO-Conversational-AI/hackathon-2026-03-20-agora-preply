- **Lower bound:** B2 - can argue, narrate, and describe with confidence; uses complex sentences; self-monitors for known errors
- **Upper bound:** C1 - hedging language, formal register, nuanced vocabulary are emerging but not yet automatic

## Gaps

### Null subject omission (Accuracy) [exercise_focus: true]
Five instances of dropping expletive "it" in embedded clauses: "I think is important" pattern. This is the single most frequent and persistent error, deeply fossilized from Spanish pro-drop. At B2, this should be largely acquired. It is the primary barrier to C1 classification.

### False friends (Range) [exercise_focus: true]
Four false friend errors (assist/attend, sensible/sensitive, comprehensive/understanding, actually/currently) plus student self-reports these as recurring. At B2, the most common Spanish-English false friends should be controlled. The self-correction on "actually" [02:16, 23:31] shows progress but the pattern remains active under pressure.

### Prepositional calques from Spanish "de" (Accuracy) [exercise_focus: true]
Four preposition errors (depend of, discussing about, interested on, dependent of). All stem from directly translating Spanish prepositions. At B2, high-frequency collocations like "depend on" and "interested in" should be acquired.

### Collocation accuracy (Range) [exercise_focus: true]
Three collocation calques (make a meeting, do a presentation, make research). These are make/do/give/take confusions rooted in Spanish "hacer" mapping. Student shows awareness and can self-correct when prompted, but errors persist in spontaneous speech.

### Progressive aspect overuse (Accuracy) [exercise_focus: false]
Two instances of progressive with stative predicates ("is being productive", "was being long"). This is a less frequent pattern and may resolve with increased exposure. Lower priority.

### Conditional structure in if-clauses (Accuracy) [exercise_focus: true]
One instance of "if I would" instead of "if I worked". Student self-corrected and demonstrated understanding of the rule when prompted, but the error appeared again later ("If I would have to choose" [15:28]). At B2, second conditional should be acquired.

## Suggestions

### For teacher
1. **[P1]** Design targeted practice for null subject insertion in embedded clauses. Use gap-fill exercises with "I think ___ is", "The problem is that ___ is" structures. This is the highest-priority fossilized error.
2. **[P1]** Create a false friends flashcard set for the 4 pairs identified (assist/attend, sensible/sensitive, comprehensive/understanding, actually/currently). Review at start of each lesson.
3. **[P2]** Collocation drills using make/do/give/take/hold/attend with business nouns. Use matching and categorization exercises.
4. **[P2]** Preposition focus: create sentences requiring "depend on", "interested in", "consist of", "independent of", "discuss [direct object]". Drill until automatic.
5. **[P3]** Continue debate-format activities - they generate rich output and surface errors under communicative pressure. Introduce more hedging and academic register.

### For student
1. **[P1]** Start the false friends journal as discussed. Add a sentence pair for each: "comprensivo = understanding, comprehensive = complete".
2. **[P1]** Before sending work emails, check for: missing "it" subjects, false friends (actually/currently, assist/attend), and collocations (make/do/give).
3. **[P2]** Record the 3-minute AI argument as homework. Listen back and count null subject omissions.
4. **[P2]** Read one Economist article daily. Underline collocations - especially make/do/give/take patterns.
5. **[P3]** Switch Freakonomics podcast to English version for commute listening.
