## Suggestions

### For teacher
1. **[P1]** Dedicate drill time to preposition collocations - create a running list (interested in, depend on, consist of, remind of, look forward to). Use gap-fill exercises with the student's own sentences.
2. **[P1]** Continue hedging practice in every lesson. Require hedged reformulations of direct statements as a warm-up activity.
3. **[P2]** Build a false-friend flashcard deck with the student. Add new entries each lesson. Test with sentence completion, not translation.
4. **[P2]** Assign informal English input (podcasts, YouTube, reality TV) and have the student report back using casual register.
5. **[P3]** Practice relative pronoun choice with sorting exercises (who for people, that/which for things).

### For student
1. **[P1]** Start a preposition notebook: for each new verb/adjective + preposition combination, write the English collocation AND the German equivalent to make the contrast explicit.
2. **[P1]** Practice hedging daily: when expressing opinions in English (even in writing), use at least one modal verb and one distancing phrase.
3. **[P2]** Watch one episode of a British TV show per week and note 5 casual discourse markers (thing is, mind you, reckon, fair enough, to be honest).
4. **[P2]** Make flashcards for false friends with example sentences, not just word pairs. Test in context.
5. **[P3]** Read 3 Guardian articles per week and underline hedging phrases. This builds passive recognition that supports active production.
